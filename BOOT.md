# BOOT.md — מה לעשות בכל הפעלה מחדש

**כל פעם שהמערכת עולה, קרא את הקבצים הבאים לפי הסדר לפני שעונה לאיש:**

## 1. זהות ואופי
- `SOUL.md` — איך להתנהג, גבולות, מדיניות אישורים עם אבי
- `IDENTITY.md` — מי אני (שם: מיילו 🐾)
- `USER.md` — מי אבי, אזור זמן, וואטסאפ

## 2. קונטקסט משפחה וקבוצה
- `group_family_context.md` — מיפוי בני המשפחה, JID הקבוצה, כללי מענה, **מטרה יומית**
- `memory/family/` — קרא את הקובץ הרלוונטי לפני כל שיחה עם בן/בת משפחה; עדכן אחרי שיחה משמעותית

## 3. זיכרון
- `MEMORY.md` — אינדקס לכל הזיכרונות
- `memory/` — קרא את הקובץ האחרון לפי תאריך (פורמט `YYYY-MM-DD.md`)

## 4. סנק WhatsApp allowFrom
בדוק שכל המספרים מ-`group_family_context.md` (כולל אבי) מופיעים ב-`channels.whatsapp.allowFrom` וב-`channels.whatsapp.groupAllowFrom` בקונפיג.
אם חסר מספר — הוסף אותו באמצעות `config.patch` לפני שממשיכים.

- עדכן את `memory/YYYY-MM-DD.md` של היום אם יש נתונים חדשים מהסשן הקודם
- **אל תשלח הודעה בוואטסאפ על הסטארטאפ** — אלא אם BOOT.md מכיל הוראה מפורשת לכך
