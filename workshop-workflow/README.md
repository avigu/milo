# Transparent Workshop Workflow

Status: ready-to-use draft package for Milo + agents + Slack + Drive.

This package defines a conversation-driven workflow where:
- Slack is the live workshop engine
- Drive is the structured state/documentation layer
- Milo is the orchestrator and rule-enforcer

Files:
- `protocol.md` — the operating model
- `stage-markers.md` — workflow stages and ownership rules
- `slack-thread-template.md` — how a task thread should begin and evolve
- `drive-task-folder-template.md` — Drive folder structure per task
- `status-doc-template.md` — the status doc template kept in Drive
- `message-markers.md` — standard inline markers like QUESTION_FOR_AVI / DECISION / BLOCKER
