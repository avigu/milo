# Drive Task Folder Template

Per task, create one folder in Drive.

Suggested structure:
- `00-status`
- `01-prd`
- `02-architecture`
- `03-execution-plan`
- `04-decisions`
- `05-links-and-artifacts`

## File purposes
### 00-status
Live structured status snapshot:
- current stage
- current owner
- blockers
- open questions
- last updated
- next step

### 01-prd
The working PRD for the task.

### 02-architecture
Architecture notes, reviews, boundaries, tradeoffs.

### 03-execution-plan
Concrete implementation sequence after PRD + architecture are locked.

### 04-decisions
Short log of decisions made during the workshop.

### 05-links-and-artifacts
Links to relevant threads, repos, docs, test results, screenshots, etc.

## Rule
Drive reflects and organizes the workshop state, but does not replace the Slack thread.
